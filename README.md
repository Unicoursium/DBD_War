# DBD War Simulator

一个 2D 杀手自动对战模拟器。地图使用 `assets/maps/normal.png`，世界坐标固定为 `1440 x 1440`，浏览器只负责缩放显示。

## 当前角色

- Trapper：默认 200 速度反弹移动，碰撞造成伤害，默认每 8 秒原地设置捕兽夹。
- Hillbilly：默认 200 速度反弹移动，冷却结束后蓄力并以默认 2000 速度沿锁定方向冲刺；冲刺穿过敌人时造成伤害并继续前进，撞墙后眩晕。

## 运行

```powershell
python -m http.server 5178 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:5178
```

页面刚打开时模拟不会自动开始，但会显示双方杀手的初始位置。点击“开始模拟”开始第一局；点击“重新模拟”会按当前输入重新开一局；点击“重置”会清空输入并回到默认静止预览。

地图位于中间的 `2:3` 蓝色背景中，地图和背景边缘默认预留 `130px`，可在左侧“地图边距”中调整。

## 输入规则

左右两侧文本框为空时使用默认值；填写数字后使用玩家输入值。初始角度为空时随机。

## 当前判定

- 捕兽夹只检测自身图片中心 50% 的矩形区域。
- 杀手踩夹只检测杀手图片下半 20% 的矩形区域。
- 杀手互相碰撞使用图片中心 65% 的矩形区域，图片四边各裁掉 17.5%。
- Trapper 放下的捕兽夹默认 5 秒内不会夹住自己，可在左侧调整；但可以立刻夹住其他杀手。
- 伤害数字显示在受伤杀手图片下方，并在 1 秒内向上飘动。
- Hillbilly 蓄力路径宽度为当前贴图对角线的 70%，路径会画到地图外并由地图画布裁切。
- Hillbilly 蓄力默认最后 0.4 秒锁定方向，可在右侧调整；锁定后不再继续追踪目标。
